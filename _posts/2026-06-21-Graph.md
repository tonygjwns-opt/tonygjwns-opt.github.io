그래프는 점(vertex)과 연결(edge)이다. 대부분의 문제는 표현(인접 리스트) → 순회(DFS/BFS) → 최단경로(Dijkstra)로 풀린다.

## 기본·표현

본질: vertex(node) + edge(연결). path = edge로 이어진 점들. 연결이 없으면 disconnected.

- directed(방향 있음), weighted(가중치 — 간선 수 최단 ≠ 비용 최소).
- 표현: 인접 행렬(간선 조회 O(1)이나 공간 O(V²), 밀집 그래프용) vs 인접 리스트(희소 그래프·순회에 효율적, 보통 이걸 씀).

```python
from collections import defaultdict
graph = defaultdict(list)
for u, v in edges:
    graph[u].append(v)
    graph[v].append(u)   # 무방향이면 양쪽에
```

## 순회 (DFS / BFS)

- **DFS**: 한 경로를 끝까지 파고든다. 경로 존재 확인에 적합(최단 보장 X). stack 또는 재귀.
- **BFS**: 원점에서 가까운 순(레벨 단위)으로 확장. 간선 수 기준 최단 경로에 적합. queue.
- 사이클이 있으면 무한 루프 → `visited` 필수. (visited=방문 기록 vs queue=방문 예정, 역할이 다름.)

```python
from collections import deque
def bfs(start):
    visited = {start}
    q = deque([start])
    while q:
        u = q.popleft()
        for v in graph[u]:
            if v not in visited:
                visited.add(v)      # 큐에 넣을 때 방문 표시(중복 삽입 방지)
                q.append(v)
```

- DFS 방문 순서: preorder(넣기 전 출력) / postorder(pop 뒤 출력) / **reverse post-order = 위상정렬**(Topological Sort).

## 최단경로

세 알고리즘을 상황에 맞게 고른다: 음수 없으면 Dijkstra, 음수 간선 있으면 Bellman-Ford, 모든 쌍이 필요하면 Floyd-Warshall.

### Dijkstra — 음수 없는 단일 시작점 (가장 빠름)

본질: 시작점에서 각 점까지 최단거리. min-heap에서 최단 점을 꺼내 인접 점을 갱신(relax). **음수 간선 불가.** O((V+E) log V).

```python
import heapq
def dijkstra(start, n):
    dist = [float('inf')] * n
    dist[start] = 0
    pq = [(0, start)]                 # (거리, 노드)
    while pq:
        d, u = heapq.heappop(pq)
        if d > dist[u]: continue      # 이미 더 짧게 처리된 노드는 skip
        for v, w in graph[u]:         # graph[u] = [(이웃, 가중치), ...]
            if d + w < dist[v]:
                dist[v] = d + w
                heapq.heappush(pq, (dist[v], v))
    return dist
```

### Bellman-Ford — 음수 간선 허용

Dijkstra는 "한 번 최단으로 확정한 노드는 다시 안 본다"는 전제로 도는데, 음수 간선이 있으면 나중에 그 확정값을 더 줄일 수 있어 틀린다. Bellman-Ford는 모든 간선을 V−1번 반복해서 relax하므로 이 문제가 없고, 덤으로 음수 사이클도 탐지한다. O(V·E).

```python
def bellman_ford(edges, n, start):     # edges = [(u, v, w), ...]
    dist = [float('inf')] * n
    dist[start] = 0
    for _ in range(n - 1):             # 최단경로는 간선을 최대 n-1개 지나므로 n-1번이면 수렴
        for u, v, w in edges:
            if dist[u] + w < dist[v]:
                dist[v] = dist[u] + w  # relax
    for u, v, w in edges:              # n번째에도 더 줄어들면
        if dist[u] + w < dist[v]:
            return None                # 음수 사이클 존재
    return dist
```

### Floyd-Warshall — 모든 쌍의 최단거리

한 시작점이 아니라 모든 정점 쌍 i→j의 최단거리가 한꺼번에 필요할 때 쓴다(정점 수가 적을 때, 대략 V≤수백). "중간 정점으로 k까지 써도 될 때 i→j가 더 짧아지나?"를 k를 하나씩 늘려가며 갱신하는 DP다. O(V³).

```python
# dist[i][j] 초기화: 자기 자신 0, 간선 있으면 가중치, 없으면 inf
for k in range(n):                 # 경유 허용 정점을 하나씩 늘림 (반드시 가장 바깥 루프)
    for i in range(n):
        for j in range(n):
            if dist[i][k] + dist[k][j] < dist[i][j]:
                dist[i][j] = dist[i][k] + dist[k][j]
```
