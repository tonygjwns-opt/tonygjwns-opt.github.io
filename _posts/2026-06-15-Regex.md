## 본질

문자열이 주어진 **패턴**에 맞는지 확인한다. literal(정확히 일치)부터 긴 문자열 속 포함 여부까지. (여기서 '문자'는 알파벳뿐 아니라 숫자 등 포함.)

## 파이썬 기본

```python
import re
korean = re.compile(r"[가-힣]")   # 한글 패턴 (컴파일해두면 재사용 빠름)
re.search(패턴, 문자열)            # 있으면 match 객체, 없으면 None (bool로도 판정)
```

## 자주 쓰는 함수

- `re.search(p, s)` : 문자열 **어디든** 첫 매칭. 있으면 match 객체, 없으면 None.
- `re.match(p, s)` : 문자열 **맨 앞**부터 매칭(앞에서 안 맞으면 None).
- `re.fullmatch(p, s)` : 문자열 **전체**가 일치해야 함.
- `re.findall(p, s)` : 매칭된 것 **전부 리스트**로(그룹 있으면 그룹만 반환).
- `re.finditer(p, s)` : 매칭들을 match 객체 이터레이터로(위치 등 필요할 때).
- `re.sub(p, repl, s)` : 매칭을 `repl`로 치환. `re.split(p, s)` : 패턴으로 분리.
- match 객체: `m.group()` / `m.group(1)` / `m.groups()`로 캡처 꺼냄. `m.start()` / `m.end()` 위치.
- 컴파일 플래그: `re.I`(대소문자 무시), `re.M`(멀티라인 `^$`), `re.S`(`.`이 줄바꿈도 매칭). 예: `re.compile(p, re.I)`.

> JS의 `/패턴/g`(global)는 Python에 없다 — 전체 매칭은 `findall`/`finditer`로 한다.

## 메타문자 / 패턴

1. `|` : 또는. `(banana|apple)` → "I like banana" / "I like apple" 둘 다.
2. `[]` : 안의 글자 중 **하나**. `[chr]at` → cat/hat/rat. `con[sc]en[sc]us` → 각 자리에 s나 c 하나. `[^...]` : 안의 것이 **아닌** 한 글자.
3. `.` : 아무 글자 하나. `...` = 아무 3글자. 진짜 점은 `\.`.
4. 범위 `[a-f]`, `[A-Za-z]`(영문 대소). 단축: `\w` = `[A-Za-z0-9_]`, `\d` = `[0-9]`, `\s` = whitespace. 여집합(대문자): `\D \W \S`.
5. `{}` : 반복 횟수. `\w{3}` 정확히 3개, `\w{4,7}` 4~7개.
6. `?` : 0번 또는 1번. 진짜 `?`는 `\?`.
7. `*` `+` : `*` 0번 이상, `+` 1번 이상. `m*e` → e/me/mee…, `m+e`는 e 불가.
8. `^` `$` : 줄 시작/끝 고정. `^aaa$` = 'aaa' 정확히. 진짜 기호는 `\^` `\$`.
9. `\b` `\B` : 단어 경계 / 비경계. `\t \n \r` : 탭·줄바꿈·캐리지리턴.
10. 그룹: `(abc)` 캡처그룹, `\1` 역참조(앞서 잡은 것과 동일 반복), `(?:abc)` 비캡처 그룹.
11. lookaround: `(?=abc)` 긍정 전방탐색, `(?!abc)` 부정 전방탐색.
12. lazy(최소 매칭): `a*?` `a+?` `a{2,}?` — 가능한 적게 매칭(기본은 greedy 최대).
13. 플래그(JS 형식): `/패턴/g` — `/`로 감싸고 끝에 플래그. `g`=global. 예: `/([A-Z])\w+/g`. (Python은 위 함수·플래그로 대신.)

## 활용 예 (한글 들어간 컬럼에서 `_` 앞부분만 추출)

```python
col_name = [i.split("_")[0] if re.search(korean, i.split("_")[0]) else i
            for i in list(time_data.columns)]
```
